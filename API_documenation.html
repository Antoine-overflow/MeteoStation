Data

Up 
get /data/measure/{date}
All measures specified from the starting date ,if there is only one date, and if there 2 dates, between the dates. (dataMeasureDateGet)
For valid response use TimeStamp format for date. Separate dates by a ','.

Path parameters

date (required)
Path Parameter — the date we want format: TimeStamp

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • json

Responses

200

successful operation

400

Invalid date supplied

404

Measure not found
Up 
get /data/{measure}
All measures specified from the sensor (dataMeasureGet)
For valid response try measures separated by ','. The valid measures are Temperature, Hygrometry, Pressure, RainFall, Brightness, WindDirection, WindVelocity, GPSPosition

Path parameters

measure (required)
Path Parameter — measure(s) that we want format: String

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • json

successful operation

400

Invalid measure supplied

404
